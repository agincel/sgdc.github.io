Template Event Page


This is where a description of an event goes, if there is no external site to link to. Game Jams will usually link to their itch.io pages, but events like Learn Jam may need these custom html pages to provide information and download links. You can always provide a redirect to these pages if needed. Copy a folder and use it with its index.html to make links easier to read (ie /events/learnjam instead of /events/learnjam.html)
