Learn Jam 2017

Learn Jam 2017 is your chance to test the waters of Game Development, with SGDC's helping hand. We'll be going through multiple engines and tools, exploring the basics of working in each of them. If you are interested in attending one of the sessions and plan on following along, we do request that you download and install the programs of interest to your computer before the event -- these are some big files.

Learn Jam will be taking place in Babbio Center Room 104 on Saturday, September 9th..

Lesson schedule is as follows. If you are interested in an event, please follow the links provided and install the latest version to your computer.

  • Unreal Engine 4: 12:00 - 1:00
    • requires account creation
  • Unity 5: 1:15 - 2:15
  • Krita (art): 2:30 - 3:15
  • BFXR (sound effects): 3:15 - 3:30
  • Game Maker Studio 2: 3:45 - 4:45
    • requires account creation
    • get the GMS2 Trial
    • we will be talking about the limitations of the trial during the event
  • Git: 5:00 - 5:30
    • Default settings on installation should be fine
    • If you're not sure download the installer and we'll install it at the event

Pizza will be provided. We hope you enjoy!

Want to do some early studying? Check out last year's tutorials on YouTube!
