Learn Jam 2018

Learn Jam 2018 is your chance to test the waters of Game Development, with SGDC's helping hand. We'll be going through multiple engines and tools with step-by-step tutorials and personal assistance, exploring the basics of working in each of them. If you are interested in attending one of the sessions and plan on following along, we do request that you download and install the programs of interest to your computer before the event -- these are some big files.

Learn Jam will be taking place in Babbio Center Room 122 on Sunday, September 16th..

Lesson schedule is as follows. If you are interested in an event, please follow the links provided and install the latest version to your computer.

  • Git: 12:00 - 12:20
    • Default settings on installation should be fine
    • If you're not sure download the installer and we'll install it at the event
  • Game Maker Studio 2: 12:20 - 1:05
    • requires account creation
    • get the GMS2 Trial
    • we will be talking about the limitations of the trial during the event
  • Krita (pixel art): 1:10 - 1:55
  • Unity 2017.1: 2:00 - 2:50
  • Blender (3D models): 3:00 - 3:50
  • Unreal Engine 4: 4:00 - 4:50
    • requires account creation

Pizza, drinks, and snacks will be provided! We hope you enjoy!

Want to do some early studying? Check out last year's tutorials on YouTube!
